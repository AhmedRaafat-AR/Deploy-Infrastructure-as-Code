# Network Infrastructure as Code
Deploy the following resources
- Virtual Private Cloud and subnets
- Internet gateway and NAT gateway
- Route table
- Export the stack output

# Server Infrastructure as Code
- Security groups
- AutoScaling group
- IAM Role
- Launch configuration
- Load balancer
- Export the stack output
